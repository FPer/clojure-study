```
x = {1, 2, 3}
y = {1, 2, 3}
z = {1, 2, 3}
```

における[x y z]の直積

```
[1 1 1] 
[1 1 2] 
[1 1 3] 
[1 2 1] 
[1 2 2] 
[1 2 3] 
[1 3 1] 
[1 3 2] 
[1 3 3] 
[2 1 1] 
[2 1 2] 
[2 1 3] 
[2 2 1] 
[2 2 2] 
[2 2 3] 
[2 3 1] 
[2 3 2] 
[2 3 3] 
[3 1 1] 
[3 1 2] 
[3 1 3] 
[3 2 1] 
[3 2 2] 
[3 2 3] 
[3 3 1] 
[3 3 2] 
[3 3 3]
```

と組み合わせ

```
(1 1 1)
(1 1 2)
(1 1 3)
(1 2 2)
(1 2 3)
(1 3 3)
(2 2 2)
(2 2 3)
(2 3 3)
(3 3 3)
```

を生成するプログラムを書いてみてください。
- リストとかベクタとかはどっちでも良い
- 順不同で構わない
